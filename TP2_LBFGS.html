def ls_wolfe(x,function,step,descent,f,df) :
    
    e1 = 1e-4
    e2 = 0.9
    s_ = 0
    sp = np.inf

    step2 = step   
    x2 = x + step2 * descent
    
    f2 = function.value(x2)
    
    while f2 >= f + e1 * step2 * np.dot(df, descent):
        sp =step2
        step2 = (s_ + step2) / 2
        x2 = x + step2 * descent
        f2 = function.value(x2) 


    while np.dot(function.grad(x2), descent) <= e2 * np.dot(df, descent):
        s_ = step2
        if sp == np.inf:
            step2 = 2 * step2
        else:
            step2 = (step2 + sp) / 2
        x2 = x + step2 * descent
        f2 = function.value(x2) 


    df2 = function.grad(x2)
    

    return x2,f2,df2,step2


def ls_wolfe_step_is_one(x,function,step,descent,f,df) :
    return ls_wolfe(x,function,1.,descent,f,df)

L-BFGS

Nous allons nous intresser l'algorithme Limited Memory BFGS. Cet algorithme est du type BFGS, c'est dire qu'il estime l'inverse de la Hessienne de $f$. Le L dans le nom de l'algorithme signifie qu'il est mmoire limite, c'est dire qu'il ne garde en mmoire que les $L$ dernires itrations de calcul pour estimer la Hessienne. L'algorithme est le suivant : Nous sommes l'itration $k$, nous notons $x_k$ l'itr et nous avons stock les vecteurs suivants pour tout $k_{min}\le i\le k$. $$ s_i=x_{i}-x_{i-1} \text{ et } y_i=\nabla f(x_{i}) -\nabla f(x_{i-1})$$ Et on a aussi stock $\rho_i=\frac{1}{(s_i,y_i)}$. Tous les $\rho_i$ doivent tre positifs. L'algorithme est le suivant

$r=-\nabla f(x_k)$

Pour $i=k,k-1,\dots k_{min}$

$\alpha_i=\rho_i(s_i \cdot r)$

$r=r-\alpha_i y_i$

$r=\displaystyle \frac{(s_{k_{min}}\cdot y_{k_{min}})}{(y_{k_{min}}\cdot y_{k_{min}})}r$ Pour $i=k_{min},k_{min}+1,\dots,k$ //

;$\beta_i=\rho_i(y_i\cdot r)$

$r=r+(\alpha_i-\beta_i)s_i$

Class BFGS

Crez une classe BFGS ci dessous, sa fonction __init__ sera de la forme __init__(self,nb_stock_max=8) o nb_stock_max est le nombre maximum d'itrations prises en compte. Cette fonction crera aussi une liste vide appele stock qui conserve les $s_i,g_i,\rho_i$. Elle devra aussi crer une liste vide nomme last_iter.

class BFGS() :
    def __init__(self,nb_stock_max=8,verbose=True) :
        self.nb_stock_max=nb_stock_max
        self.stock = []
        self.last_iter =[]



    def push(self, x, g):


        if self.last_iter != [] :
            s = x - self.last_iter[0]
            y = g - self.last_iter[1]
            rho = 1.0 / (np.dot(y, s))
            if rho >0 : 
                if len(self.stock) == self.nb_stock_max :
                    self.stock.pop(0)


                self.stock.append([s,y,rho])

            else :
                self.stock = []

        self.last_iter = [x,g]

        

    def get(self, grad) :
        
        
        q = -grad
        if len(self.stock) == 0 :
           pass
        else : 
            L_a = []

            for s,y,rho in reversed(self.stock) :
                alpha = rho * np.dot(s,q)
                L_a.append(alpha)
                q = q - alpha * y

            L_a.reverse()

            gamma = np.dot(self.stock[-1][0], self.stock[-1][1]) / np.dot(self.stock[-1][1], self.stock[-1][1])

            B0 = gamma * np.ones(len(grad))

            q = B0 * q

            for (s, y, rho), alpha in zip(self.stock, L_a):
                beta = rho * np.dot(y, q)
                q = q + s * (alpha - beta)


        return q


        
        
            
            





    def dc(self,x,function,df) :
        self.push(x,df)
        return self.get(df)

Push

Nous allons maintenant crer une fonction push(self, x, grad) qui enregistre $s_k,g_k,\rho_k$. Pour cela, on a besoin de $x_{k-1},\nabla f(x_{k-1})$. Si ils existent, ils se trouvent dans la liste self.last_iter. Ensuite on peut calculer $s_k,g_k$ et $\rho_k$.

Si $\rho_k$ est positif, alors on enregistre le triplet $(s_k,g_k,\rho_k)$ la fin de la liste self.stock, en vrifiant self.stock ne doit contenir au maximum que les dernires self.nb_stock_max itrations (si ncessaire on retire le tout premier lment de self.stock).

Si $\rho_k$ est ngatif, quelquechose c'est mal pass, on vide le self.stock.

A la fin, on n'oublie pas de mettre $x_{k}$ et $\nabla f(x_{k})$ dans self.last_iter pour tre sr de les y trouver la prochaine fois.

Get

Nous allons maintenant crer une fonction get(self, grad) qui modifie la direction de descente et applique l'algorithme ci-dessus. Cette fonction doit nous rendre le r final. Si le self.stock est vide, cette fonction doit nous rendre -grad

dc

Nous creons maintenant une fonction dc(self,x,function,df) qui appplique tout d'abord self.push puis self.get, elle rend le rsultat de la fonction de self.get.
